## 💡 QUICK START (5 MINUTES FOR EXPERIENCED DEVOPS)

```bash
# 1. Backup
cp database/database.sqlite database/database_backup_$(date +%s).sqlite

# 2. Setup MySQL (assuming already installed)
mysql -u root -p < database/mysql_setup.sql

# 3. Configure  
nano .env  # Uncomment MySQL, comment SQLite

# 4. Migrate
php artisan migrate:fresh
php database/migrate_sqlite_to_mysql.php
php database/validate_migration.php

# 5. Test
php artisan serve --host=0.0.0.0 --port=8000
# Visit http://192.168.1.4:8000 ✅
```
