## 📝 Yang Perlu Anda Lakukan

### **Step 1: Install MySQL Server** (Hanya sekali)
```
1. Download MySQL: https://dev.mysql.com/downloads/mysql/
2. Install MySQL Server 8.0 (Community Edition)
3. Catat MySQL root password
4. Test: mysql --version
```

### **Step 2: Run Setup Script**
```bash
mysql -u root -p < database/mysql_setup.sql
```
Masukkan root password MySQL Anda

### **Step 3: Test MySQL Connection**
```bash
php artisan tinker
>>> DB::connection('mysql')->select('SELECT 1')
```

### **Step 4: Create Tables di MySQL**
```bash
php artisan migrate:fresh
```

### **Step 5: Migrate Data dari SQLite**
```bash
php database/migrate_sqlite_to_mysql.php
```

### **Step 6: Restart Server & Verify**
```bash
php artisan serve --host=0.0.0.0 --port=8000
```
