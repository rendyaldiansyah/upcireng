  Issue #1: No Sound on New Order
  Fix:
    1. Check browser volume not muted 🔕
    2. Check sound file exists: public/sounds/chord.mp3
    3. Open browser console (F12)
    4. Look for errors starting with [ORDER NOTIFICATIONS]
    5. Test manually: window.realtimeNotifications.playSound()

  Issue #2: WebSocket Not Connecting
  Fix:
    1. Check Reverb running: php artisan reverb:start
    2. Check port 8080 not blocked by firewall
    3. Console check: typeof window.Echo should be "object"
    4. Fallback will activate (polling every 5s)

  Issue #3: Sound Plays on Every Refresh
  Fix:
    1. Clear localStorage: localStorage.clear()
    2. Refresh page
    3. Create new order
    4. Anti-spam should now work

  Issue #4: QRIS Image Not Showing
  Fix:
    1. Admin upload QRIS in Settings
    2. Run: php artisan storage:link
    3. Check file exists: storage/app/public/qris/filename.jpg
    4. Check permissions: chmod 755 storage/
